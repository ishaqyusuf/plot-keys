# Product Roadmap

## Purpose
This file tracks the likely implementation sequence at a product level.

## How To Use
- Keep entries outcome-oriented.
- Move items to task files when they become actionable engineering work.

## Phase 0: Foundation
- Initialize monorepo structure
- Establish shared UI package and design tokens
- Set up tenant-aware app shells
- Define initial auth, database, and billing approach

## Phase 1: Core Operations
- Company management
- User and agent management
- Property listings
- Property media
- Lead capture
- Appointment scheduling

## Phase 2: Website Generator
- Template system
- Template management foundation
- Theme config
- Template config mode with fonts, color systems, style presets, and named image slots
- Section registry
- Tenant website rendering
- CMS editing for content blocks
- Draft/live publish model
- Auto-sync property listings to public site

## Phase 3: Commercial Platform
- Subscription billing
- Tiered plan entitlements for Starter, Plus, and Pro
- Paid template unlocks and entitlement tracking
- Template marketplace licensing records
- Stock image marketplace and licensing
- Custom domains
- Domain purchase and management
- SSL automation
- Analytics foundations

## Phase 4: AI Layer
- Website content generation
- Inline AI generation for editable content nodes
- Property description generation
- Blog and SEO generation
- AI usage tracking and credits
- AI billing and credit packs

## Phase 5: Premium Experience
- Pro template expansion
- Logo system and AI-assisted brand assets
- Mobile app request packaging
- Template marketplace
- Uniqueness and template popularity insights

## Phase 6: Expansion
- Construction project management foundation
- Customer project interface for active developments
- Project budgeting, QS, and site workforce workflows
- AI lead reply assistant
- AI recommendations
- AI layout suggestions
- Third-party theme developer ecosystem

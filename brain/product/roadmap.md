# Product Roadmap

## Purpose
This file tracks the likely implementation sequence at a product level.

## How To Use
- Keep entries outcome-oriented.
- Move items to task files when they become actionable engineering work.

## Phase 0: Foundation
- Initialize monorepo structure
- Establish shared UI package and design tokens
- Set up tenant-aware app shells
- Define initial auth, database, and billing approach

## Phase 1: Core Operations
- Company management
- User and agent management
- Property listings
- Property media
- Lead capture
- Appointment scheduling

## Phase 2: Website Generator
- Template system
- Theme config
- Section registry
- Tenant website rendering
- CMS editing for content blocks
- Auto-sync property listings to public site

## Phase 3: Commercial Platform
- Subscription billing
- Custom domains
- Domain purchase and management
- SSL automation
- Analytics foundations

## Phase 4: AI Layer
- Website content generation
- Property description generation
- Blog and SEO generation
- AI usage tracking and credits
- AI billing and credit packs

## Phase 5: Expansion
- Template marketplace
- AI lead reply assistant
- AI recommendations
- AI layout suggestions
- Third-party theme developer ecosystem

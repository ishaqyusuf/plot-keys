# ADR Template

## Purpose
This file is the starter template for architecture decisions.

## How To Use
- Create `ADR-XXX-<slug>.md` in `brain/decisions/`.
- Use incremental numbering.

## Template
```md
# ADR-XXX: <Decision Title>

## Status
Proposed

## Context
What problem or constraint led to this decision.

## Decision
What was chosen.

## Consequences
- Positive impact
- Tradeoff

## Alternatives Considered
- Alternative
```

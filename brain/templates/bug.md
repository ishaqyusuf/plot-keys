# Bug Template

## Purpose
This file is the starter template for bug memory records.

## How To Use
- Create bug files in `brain/bugs/` when an issue has a root cause worth preserving.

## Template
```md
# <Bug Title>

## Date
YYYY-MM-DD

## Summary
Short statement of the bug.

## Root Cause
What caused it.

## Fix
What changed.

## Prevention
How to avoid similar issues.

## Related Files
- File path
```
